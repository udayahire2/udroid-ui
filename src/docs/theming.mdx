## Radius & Spacing

Control the "roundness" of your interface globally.

```css
:root {
  --radius: 0.5rem; /* 8px */
}
```

## Dark Mode

Dark mode is first-class citizen in UDROID UI. It uses the `dark` class strategy.
By default, we use **CSS Variables** to handle color switching automatically.

```css
/* Light Mode */
:root {
  --background: 0 0% 100%;
  --foreground: 240 10% 3.9%;
}

/* Dark Mode */
.dark {
  --background: 240 10% 3.9%;
  --foreground: 0 0% 98%;
}
```

## Customizing

To customize your theme, simply edit the `src/index.css` file.
You can use `hsl` values for fine-grained control over opacity modifiers.

  --primary-foreground: 210 40% 98%;
</div>
