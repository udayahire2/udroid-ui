
# Installation

## React / Next.js

1.  Create a new project.
    ```bash
    npx create-next-app@latest my-app --typescript --tailwind --eslint
    ```

2.  Run the CLI.
    ```bash
    npx udroid-ui init
    ```

## Vite

1.  Create a new project.
    ```bash
    npm create vite@latest
    ```

2.  Run the CLI.
    ```bash
    npx udroid-ui init
    ```
